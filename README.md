- 개발 기간 : 2023/10/26 ~ 2023/11/30

## 프로잭트 개요
- 프로젝트명 : Vocational Test
- 현재 직장 만족도를 테스트해보는 웹 사이트
### ✔️ 목표
- Next.js와 Typescript로 미니 프로젝트 만들기
- 간단한 테스트 사이트 구현해보기
- 카카오 공유 기능 사용해보기

## 🏋️‍♀️ 구현 기능
- 테스트 진척도를 확인 할 수 있는 Progress Bar 구현
- Zustand를 활용한 전역 상태 관리
- 테스트 결과 카카오 공유 기능 구현
- 테스트 결과 페이지 URL 링크보드 복사 기능 구현
- Vecel을 통한 배포 완료
